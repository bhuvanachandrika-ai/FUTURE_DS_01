Future Intern Task 1 – Online Retail Sales Dashboard

Project Overview

This project analyzes online retail sales data using Power BI. 
The objective is to transform raw transaction data into meaningful business insights through data cleaning, modeling, DAX calculations, and interactive visualizations.

Dataset

The dataset contains online retail transactions with the following fields:

- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

Data Preparation

The following data preparation steps were performed:

1. Imported the dataset into Power BI.
2. Removed invalid or missing customer records.
3. Created a Revenue column using:

Revenue = Quantity × UnitPrice

4. Verified data types and cleaned the dataset.

DAX Measures

The following measures were created:

- Total Revenue
- Total Orders
- Total Customers
- Total Quantity Sold

These measures provide key business performance indicators.

Dashboard Visualizations

KPI Cards

- Total Revenue
- Total Orders
- Total Customers
- Total Quantity Sold

Revenue Trend Analysis

A line chart was created to analyze revenue performance over time.

Product Performance

A bar chart was used to identify top-selling products based on revenue.

Country Analysis

Country-wise sales performance was visualized to compare revenue across locations.

Interactive Filters

Slicers were added for:

- Country
- Year
- Month

Key Insights

- Revenue trends help identify periods of high sales activity.
- Product analysis highlights the most profitable items.
- Country analysis reveals major revenue-generating regions.
- KPI cards provide a quick summary of overall business performance.

Conclusion

The Power BI dashboard successfully converts retail transaction data into actionable insights. The report enables users to monitor sales performance, identify top products, and make data-driven business decisions through interactive visualizations.
